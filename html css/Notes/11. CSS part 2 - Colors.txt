COLOR PROPERTIES
	> property name is "color"
	> responsible for color of text

COLOR BACKGROUND-COLOR
	> property name is "background-color"
	> responsible for background color of the text

BACKGROUND-IMAGE
	> "background-image"
	> used to assign an image as a background

BACKGROUND	
	> "background"
	> can accept either a color or an image

COLOR NAMES
	> when specifying color value, we can pass these pre-defined colors with names
	> as of now, theres 140 HTML COLOR NAMES available 
	
EX:
h1{
	color:red;
}


COLOR RGB (RED, GREEN, BLUE)
	> we can assign color that is represented by three numbers
	> these numbers represents the intensity of either red, green, or blue (based on position of number)
	> you can only assign intensity number of red, green, or blue from (0-255)
	> for black, reduce the RGB to 0. for WHITE. max out the RGB to 255
		- black = rgb(0,0,0)
		- white = rgb(255,255,255)

EX:
h1{
	color:rgb(255, 0, 0);
}

	> this will be a color red



COLOR RGBA
	> same as RGB but with additional parameter (opacity/transparency)
	> the A takes a decimal number from 0 to 1
	> 0 meaning the color is transparent/invisible, 1 meaning full visibility




COLOR HSL
	> HSL stands for Hue, Saturation, Lightness
		- Hue is for what color it is
			> takes 0-360 as value
		- Saturation is how saturated the color is
			> takes 0-100 as value
		- Lightness is how bright the color is
			> takes 0-100 as value



COLOR HEX VALUE
	> format is #RRGGBB
	> uses Hexadecimal to determine the intensity of RGB	
	> lowest number is #000000 while highest is #ffffff
	> divided by 3 parts
		- first two characters defines the Red
		- second two characters defines the Green
		- third two characters defines the Blue
	> Black can be fetch by only #000 or #222
	> White can be fetch by only #fff


coolors.co
	> a website for generating random color palette

9. Inside the styles.css for each of the id attributes of h3
