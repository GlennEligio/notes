02. Maven Commands


Install Maven on Ubuntu OS

apt update
apt install maven -y


mvn --version
	> checks the Maven version installed


mvn archetype:generate
	> downloads the archetypes templates to be able to generate
	> user will then be prompted to choose number corresponding to specific archetype
	> user will be prompt to provide information regarding GAV (groupId, artifactId, version)

mvn clean
	> cleans the target folder generated by mvn compile

mvn compile
	> compile the code and create target folder

mvn package
	> to create jar/war file in the target folder

mvn <build-phase>
	> command template used to execute any point in the Maven Build phase
