Cloud Monitoring Summary

CloudWatch
 > Metrics
	- monitor the performance of AWS services and billing metrics
 > Alarms
	- automate notification, perform EC2 action, notify to SNS based on metrics
 > Logs
	- collect log files from EC2 instances, services, Lambda functions..
 > Events (or EventBridge)
	- react to events in AWS, or trigger a rule on a schedule basis
CloudTrail
 > used for auditing API calls made within your AWS account
CloudTrail Insights
 > automated analysis of your CloudTrail events
X-Ray
 > trace requests made through your distributed applications
Service Health Dashboard
 > status of all AWS services across all regions
Personal Health Dashboard
 > AWS events that impact your infrastructure
Amazon CodeGuru
 > automated code reviews (CodeReviewer) and application performance recommendations (CodeProfiler)
