07. Loops

VIII. Loops

Loops
	> used to do a function in repetitions
	> we have two generic solutions
		a. For loops
		b. While loops
	> we also have a loop specially made for iterating through an array
		b. For of loops

EX:
// Loops

// For
for (let i = 0; i <= 10; i++) {
  console.log(`For loop number: ${i}`);
}

// While
let i = 0;
while (i < 10) {
  console.log(`While loop number: ${i}`);
  i++;
}

const todos = [
  {
