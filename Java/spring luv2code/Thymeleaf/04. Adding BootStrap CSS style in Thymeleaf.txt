04. Adding BootStrap CSS style in Thymeleaf


Adding BootStrap CSS
	> either
		1. download the BootStrap css file and place in /src/main/resources/static/css
		2. add a <link> inside <head> and pass the BootStrap css file link as the "href" parameter

Link for BootStrap CSS standalone

<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.1.3/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-1BmE4kWBq78iYhFldvKuhfTAU6auU8tT94WrHftjDbrCEXSU1oBoqyl2QvZ6jIW3" crossorigin="anonymous">



BootStrap config for a more responsive page
	> add inside <head> element

    <!-- Required meta tags -->
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1">

    <!-- Bootstrap CSS -->
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.1.3/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-1BmE4kWBq78iYhFldvKuhfTAU6auU8tT94WrHftjDbrCEXSU1oBoqyl2QvZ6jIW3" crossorigin="anonymous">


Some classes in BootStrap CSS

FOR TABLES

"container"
	> added in a <div>
	> used to create a BootStrap div container

"table"
	> added in <table>
	> used to create a BootStrap table

"table-bordered"
	> added in <table>
	> used to make the table bordered

"table-striped"
	> added in <table>
	> used to make the table rows background color striped (light-darkgrey)

"thead-dark"
	> added in <thead>
	> used to make the table head dark theme (black background - white text)

"table-dark"
	> added in <table> and <thead>
	> same effect as "thead-dark", inverses the color of background and text



FOR BUTTON AND HYPERLINKS

"btn"
	> added in <a> and <button>
	> used to make a BootStrap button
	> element will have
		- rounded corner of border
	> Adds basic styling to any button

"btn-primary"
	> added in <a> and <button>
	> with this, the element will have
		- blue background
		- white text
	> Provides extra visual weight and identifies the primary action in a set of buttons

"btn-info"
	> Contextual button for informational alert messages
	> element will have
		- sky blue background
		- white text

"btn-sm"
	> used to create a small Button




FOR Input and TextArea

"form-control"
	> Give textual form controls like <input>s and <textarea>s an upgrade with custom styles, sizing, focus states, and more.





GENERAL CLASS

"m-x"
	> used to define margin
	> x can be from 1-5		
	> higher the number, larger the margin
	> this will add margin in ALL SIDE	
	> to add margin only in specific side, add 
		t on m for top margin ("mt-4")
		b for bottom margin
		l for left margin
		r for right margin


"col"
	> used for defining alignment of elements by columns
