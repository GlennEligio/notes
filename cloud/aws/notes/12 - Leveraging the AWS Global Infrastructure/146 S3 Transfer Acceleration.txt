S3 Transfer Acceleration
 > Increase transfer speed by transferring file to an AWS edge which will forward the data to the S3 bucket in the target region
 > you can test the speed gain using the tool below
	- https://s3-accelerate-speedtest.s3-accelerate.amazonaws.com/en/accelerate-speed-comparison.html

	Fast 				Fast
File ------------> Edge Location -------------->  S3 Bucket
